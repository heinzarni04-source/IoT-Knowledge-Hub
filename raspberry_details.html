🖥️ Raspberry Pi System Overview

Hardware Components

  • ARM Processor (Quad-core)
  • RAM (2GB - 8GB)
  • microSD Card Slot
  • USB Ports (2-4)
  • Ethernet Port
  • WiFi & Bluetooth

Key Features

  • Full Linux Operating System
  • 40 GPIO Pins
  • I2C, SPI, UART Support
  • Low Power Consumption
  • Cost Effective
  • Large Community Support

🔌 GPIO Pinout & Connections

⚠️ Important: Raspberry Pi GPIO pins are 3.3V logic. Connecting 5V devices directly can damage the board. Always use level shifters or voltage dividers.

GPIO Pin Categories

Power Pins

  • Pin 1, 17: 3.3V Power
  • Pin 2, 4: 5V Power
  • Pin 6, 9, 14, 20, 25, 30, 34, 39: Ground (GND)

Communication Pins

  • I2C: GPIO 2 (SDA), GPIO 3 (SCL)
  • SPI: GPIO 10 (MOSI), GPIO 9 (MISO), GPIO 11 (SCLK)
  • UART: GPIO 14 (TX), GPIO 15 (RX)

⚙️ Setup & Installation

Step 1: Prepare microSD Card

  1. Download Raspberry Pi OS from official website
  2. Use Raspberry Pi Imager to write OS to microSD card
  3. Enable SSH and WiFi in imager settings
  4. Insert microSD card into Raspberry Pi

Step 2: First Boot

  1. Connect power supply (5V 3A USB-C)
  2. Wait for LED indicators to stabilize
  3. Connect via SSH: ssh pi@raspberrypi.local
  4. Default password: raspberry

Step 3: Update System

Bash 
sudo apt update
sudo apt upgrade -y
sudo apt install python3-pip -y

🐍 Python GPIO Programming

Install Required Libraries

Bash 
pip3 install RPi.GPIO
pip3 install gpiozero

Basic LED Control Example

Python 
import RPi.GPIO as GPIO
import time

# Setup
GPIO.setmode(GPIO.BCM)
GPIO.setup(17, GPIO.OUT)

try:
    while True:
        GPIO.output(17, GPIO.HIGH)
        print("LED ON")
        time.sleep(1)
        
        GPIO.output(17, GPIO.LOW)
        print("LED OFF")
        time.sleep(1)
        
except KeyboardInterrupt:
    print("Exiting...")
finally:
    GPIO.cleanup()

Button Input Example

Python 
from gpiozero import Button, LED
from signal import pause

# Setup
led = LED(17)
button = Button(27)

# Event handlers
button.when_pressed = led.on
button.when_released = led.off

print("Press Ctrl+C to exit")
pause()
