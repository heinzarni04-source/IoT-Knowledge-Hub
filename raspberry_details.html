Raspberry Pi Full Guide

By: Hein ရေးသားသူ: Hein
Raspberry Pi is a credit-card-sized Single Board Computer (SBC) that runs a Linux-based Operating System. It can control sensors and actuators through GPIO pins and supports Python programming, making it ideal for IoT projects. Raspberry Pi ဆိုတာ အစ်ခြင်းအရွယ်ရှိတဲ့ Single Board Computer (SBC) တစ်ခုဖြစ်ပြီး Linux အခြေခံ Operating System ကို အသုံးပြုတယ်။ GPIO pins များမှတစ်ဆင့် sensors နဲ့ actuators တွေကို ထိန်းချုပ်နိုင်ပြီး Python ဖြင့် programming လုပ်လို့ရတဲ့ အတွက် IoT projects များအတွက် အလွန်နှစ်သက်ဖွယ်ရှိတယ်။

🔌 GPIO Pinout & ConnectionsGPIO Pinout & ချိတ်ဆက်မှုများ

⚠️ Important: Raspberry Pi GPIO pins are 3.3V logic. Connecting 5V devices directly can damage the board. Always use level shifters or voltage dividers. ⚠️ သတိပြုရန်: Raspberry Pi GPIO pins များသည် 3.3V logic ဖြစ်သည်။ 5V devices တိုက်ရိုက်ချိတ်ဆက်ပါက board ပျက်စီးနိုင်သည်။ Level shifters သို့မဟုတ် voltage dividers အမြဲသုံးပါ။

GPIO Pin CategoriesGPIO Pin အမျိုးအစားများ

Power Pinsပါဝါ Pins

  • Pin 1, 17: 3.3V Power
  • Pin 2, 4: 5V Power
  • Pin 6, 9, 14, 20, 25, 30, 34, 39: Ground (GND)

Communication Pinsဆက်သွယ်ရေး Pins

  • I2C: GPIO 2 (SDA), GPIO 3 (SCL)
  • SPI: GPIO 10 (MOSI), GPIO 9 (MISO), GPIO 11 (SCLK)
  • UART: GPIO 14 (TX), GPIO 15 (RX)

⚙️ Setup & Installationတပ်ဆင်ခြင်း

Step 1: Prepare microSD Cardအဆင့် ၁: microSD Card ပြင်ဆင်ရန်

  1. Download Raspberry Pi OS from official websiteတရားဝင် website မှ Raspberry Pi OS ဒေါင်းလုပ်ရန်
  2. Use Raspberry Pi Imager to write OS to microSD cardRaspberry Pi Imager ဖြင့် OS ကို microSD card သို့ ရေးသွင်းရန်
  3. Enable SSH and WiFi in imager settingsImager settings တွင် SSH နှင့် WiFi ဖွင့်ရန်
  4. Insert microSD card into Raspberry PimicroSD card ကို Raspberry Pi ထဲ ထည့်ရန်

Step 2: First Bootအဆင့် ၂: ပထမဆုံး ဖွင့်ခြင်း

  1. Connect power supply (5V 3A USB-C)Power supply (5V 3A USB-C) ချိတ်ဆက်ရန်
  2. Wait for LED indicators to stabilizeLED indicators တည်ငြိမ်သည်အထိ စောင့်ရန်
  3. Connect via SSH: SSH ဖြင့် ချိတ်ဆက်ရန်: ssh pi@raspberrypi.local
  4. Default password: Default password: raspberry

Basic LED Control ExampleLED ထိန်းချုပ်မှု နမူနာ

Python 
import RPi.GPIO as GPIO
import time

# Setup
GPIO.setmode(GPIO.BCM)
GPIO.setup(17, GPIO.OUT)

try:
    while True:
        GPIO.output(17, GPIO.HIGH)
        print("LED ON")
        time.sleep(1)
        
        GPIO.output(17, GPIO.LOW)
        print("LED OFF")
        time.sleep(1)
        
except KeyboardInterrupt:
    print("Exiting...")
finally:
    GPIO.cleanup()

💡 Pro Tips:
  • Always use pull-up/pull-down resistors for GPIO inputsGPIO inputs အတွက် pull-up/pull-down resistors အမြဲသုံးပါ
  • Use relay modules for controlling high-power devicesHigh-power devices ထိန်းချုပ်ရန် relay modules သုံးပါ
  • Consider using GPIO expanders for more pinsPins ပိုလိုပါက GPIO expanders သုံးရန် စဉ်းစားပါ
