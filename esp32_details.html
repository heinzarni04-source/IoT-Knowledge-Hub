ESP32 DevKit V1 (30-Pin) Full Guide

Developed by: Hein ရေးသားသူ: Hein
ESP32 is a low-cost, low-power "System on a Chip" (SoC) that includes Wi-Fi and Dual-mode Bluetooth connectivity, making it the most widely used microcontroller for IoT projects. ESP32 ဆိုတာ low-cost, low-power "System on a Chip" (SoC) တစ်ခုဖြစ်ပြီး Wi-Fi နဲ့ Dual-mode Bluetooth ဆက်သွယ်မှုတွေ ပါဝင်တဲ့အတွက် IoT ပရောဂျက်တွေမှာ အသုံးအများဆုံး microcontroller ဖြစ်ပါတယ်။

ESP32 Pinout Diagram

1. Pin Layout Details ၁။ Pin Layout အသေးစိတ်

Power Pins ပါဝါငုတ်များ (Power Pins)

  • VIN: 5V External Power supply. 5V External Power ပေးရန်။
  • 3V3: 3.3V Output (for Sensors). 3.3V Output (Sensor များအတွက်)။
  • GND: Ground (negative terminal). Ground (အနှုတ်ငုတ်)။

Communication Pins ဆက်သွယ်ရေး (Communication)

  • I2C: SDA (GPIO 21), SCL (GPIO 22)
  • SPI: MOSI(23), MISO(19), SCK(18), CS(5)
  • UART: TX0/RX0 & TX2/RX2 (GPIO 17/16)
⚠️ Warning: ESP32 GPIO pins are 3.3V Logic only. Applying 5V may burn the chip. ⚠️ သတိပြုရန်: ESP32 GPIO များသည် 3.3V Logic သာဖြစ်ပြီး 5V ပေးမိပါက Chip လောင်သွားနိုင်ပါသည်။

2. Special Attention Pins ၂။ အထူးသတိထားရမည့် Pins များ

Pin Name Category Details အချက်အလက်
GPIO 0 Boot Connects to Ground → enters Programming Mode. Ground နှင့်ထိထားလျှင် Programming Mode ထဲရောက်မည်။
GPIO 34, 35, 36, 39 Input Only Cannot be used as Output. Output အဖြစ်သုံး၍ မရပါ။
GPIO 6 to 11 Flash Memory Connected to Internal Flash — do not use. Internal Flash နှင့် ချိတ်ထားသဖြင့် လုံးဝမသုံးသင့်ပါ။
GPIO 1 & 3 Serial Used for Hardware Serial — not suitable for I/O. Hardware Serial အတွက်ဖြစ်၍ I/O အဖြစ်သုံးရန်မသင့်ပါ။

3. Practical Code Example ၃။ လက်တွေ့စမ်းသပ်ရန် Code Example

First test: Serial Monitoring and LED Blink code. ပထမဆုံး စမ်းသပ်ရန် Serial Monitoring နှင့် LED Blink ကုဒ်ဖြစ်ပါသည်။

Arduino IDE (.ino) 
#define LED_PIN 2

void setup() {
  Serial.begin(115200);
  pinMode(LED_PIN, OUTPUT);
}

void loop() {
  Serial.println("LED is ON");
  digitalWrite(LED_PIN, HIGH);
  delay(1000);

  Serial.println("LED is OFF");
  digitalWrite(LED_PIN, LOW);
  delay(1000);
}
💡 Pro Tip:
If Upload Error occurs, hold the BOOT button. Upload Error တက်လျှင် BOOT button ကို ဖိထားပေးပါ။
🔌 Driver:
If port not found, install CP2102 Driver. Port မပေါ်ပါက CP2102 Driver ကို Install လုပ်ပါ။
