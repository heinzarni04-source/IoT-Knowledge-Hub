ESP32 DevKit V1 (30-Pin) Full Guide

ESP32 ဆိုတာ low-cost, low-power "System on a Chip" (SoC) တစ်ခုဖြစ်ပြီး Wi-Fi နဲ့ Dual-mode Bluetooth ဆက်သွယ်မှုတွေ ပါဝင်တဲ့အတွက် IoT ပရောဂျက်တွေမှာ အသုံးအများဆုံး microcontroller ဖြစ်ပါတယ်။

ESP32 Pinout Diagram

၁။ Pin Layout အသေးစိတ်

ပါဝါငုတ်များ (Power Pins)

  • VIN: 5V External Power ပေးရန်။
  • 3V3: 3.3V Output (Sensor များအတွက်)။
  • GND: Ground (အနှုတ်ငုတ်)။

ဆက်သွယ်ရေး (Communication)

  • I2C: SDA (GPIO 21), SCL (GPIO 22)
  • SPI: MOSI(23), MISO(19), SCK(18), CS(5)
  • UART: TX0/RX0 နှင့် TX2/RX2 (GPIO 17/16)
⚠️ သတိပြုရန်: ESP32 GPIO များသည် 3.3V Logic သာဖြစ်ပြီး 5V ပေးမိပါက Chip လောင်သွားနိုင်ပါသည်။

၂။ အထူးသတိထားရမည့် Pins များ

Pin Name Category အချက်အလက်
GPIO 0 Boot Ground နှင့်ထိထားလျှင် Programming Mode ထဲရောက်မည်။
GPIO 34, 35, 36, 39 Input Only Output အဖြစ်သုံး၍ မရပါ။
GPIO 6 to 11 Flash Memory Internal Flash နှင့် ချိတ်ထားသဖြင့် လုံးဝမသုံးသင့်ပါ။
GPIO 1 & 3 Serial Hardware Serial အတွက်ဖြစ်၍ I/O အဖြစ်သုံးရန်မသင့်ပါ။

၃။ လက်တွေ့စမ်းသပ်ရန် Code Example

ပထမဆုံး စမ်းသပ်ရန် Serial Monitoring နှင့် LED Blink ကုဒ်ဖြစ်ပါသည်။

Arduino IDE (.ino) 
#define LED_PIN 2

void setup() {
  Serial.begin(115200);
  pinMode(LED_PIN, OUTPUT);
}

void loop() {
  Serial.println("LED is ON");
  digitalWrite(LED_PIN, HIGH);
  delay(1000);

  Serial.println("LED is OFF");
  digitalWrite(LED_PIN, LOW);
  delay(1000);
}
