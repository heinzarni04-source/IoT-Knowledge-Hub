⚙️ Arduino IDE SetupArduino IDE တပ်ဆင်ခြင်း

Step 1: Download & Installအဆင့် ၁: ဒေါင်းလုပ် & တပ်ဆင်ရန်

  1. Visit Arduino Official WebsiteArduino တရားဝင် Website သွားရောက်ပါ
  2. Download Arduino IDE 2.0 or laterArduino IDE 2.0 သို့မဟုတ် နောက်ပိုင်း ဒေါင်းလုပ်ပါ
  3. Install on your computer (Windows, Mac, or Linux)ကွန်ပျူတာတွင် တပ်ဆင်ပါ (Windows, Mac, သို့မဟုတ် Linux)
  4. Launch the IDEIDE ဖွင့်ပါ

Step 2: Install Board Supportအဆင့် ၂: Board Support တပ်ဆင်ရန်

  1. Go to Tools → Board → Board ManagerTools → Board → Board Manager သွားပါ
  2. Search for your board (e.g., "Arduino AVR")မိမိ board ကို ရှာပါ (ဥပမာ "Arduino AVR")
  3. Click InstallInstall နှိပ်ပါ
  4. Select your board from Tools → BoardTools → Board မှ မိမိ board ကို ရွေးပါ

Step 3: Select Port & Uploadအဆင့် ၃: Port ရွေးပြီး Upload လုပ်ရန်

Stepsအဆင့်များ 
1. Connect Arduino via USB
2. Tools → Port → Select COM port
3. Tools → Board → Select your board
4. Sketch → Upload (or Ctrl+U)

🔧 Arduino Programming BasicsArduino Programming အခြေခံများ

Basic Structureအခြေခံ ဖွဲ့စည်းပုံ

Arduino (C++) 
// This function runs once when powered on
void setup() {
  Serial.begin(9600);  // Initialize serial communication
  pinMode(13, OUTPUT); // Set pin 13 as output
}

// This function runs repeatedly
void loop() {
  digitalWrite(13, HIGH);  // Turn LED on
  delay(1000);             // Wait 1 second
  
  digitalWrite(13, LOW);   // Turn LED off
  delay(1000);             // Wait 1 second
}

Digital I/O ExampleDigital I/O နမူနာ

Arduino (C++) 
// LED Blink with Button
const int ledPin = 13;
const int buttonPin = 2;

void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(buttonPin, INPUT);
}

void loop() {
  if (digitalRead(buttonPin) == HIGH) {
    digitalWrite(ledPin, HIGH);
  } else {
    digitalWrite(ledPin, LOW);
  }
}

📊 Working with SensorsSensors နှင့် အလုပ်လုပ်ခြင်း

Analog Input Example (Temperature Sensor)Analog Input နမူနာ (အပူချိန် Sensor)

Arduino (C++) 
const int sensorPin = A0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int sensorValue = analogRead(sensorPin);
  float voltage = sensorValue * (5.0 / 1023.0);
  float temperature = (voltage - 0.5) * 100;
  
  Serial.print("Temperature: ");
  Serial.print(temperature);
  Serial.println(" C");
  
  delay(1000);
}

💡 Best Practices:အကောင်းဆုံး လုပ်ဆောင်နည်းများ:
  • Always use appropriate resistors and capacitorsသင့်တော်သော resistors နှင့် capacitors အမြဲသုံးပါ
  • Test code in simulation before uploadingUpload မတင်မီ simulation တွင် code စမ်းသပ်ပါ
  • Use meaningful variable namesအဓိပ္ပာယ်ရှိသော variable names သုံးပါ
  • Add comments to explain code logicCode logic ရှင်းပြရန် comments ထည့်ပါ
  • Check voltage levels before connecting componentsComponents ချိတ်ဆက်မတင်မီ voltage levels စစ်ဆေးပါ
  • Use Serial Monitor for debuggingDebug လုပ်ရန် Serial Monitor သုံးပါ
⚠️ Safety Tips:
  • Never connect 5V directly to analog inputs (max 5V)
  • Use current-limiting resistors with LEDs
  • Disconnect power before making circuit changes
  • Use USB power for safe operation
 ⚠️ လုံခြုံရေး အကြံပြုချက်များ:
  • Analog inputs သို့ 5V တိုက်ရိုက် မချိတ်ဆက်ပါနှင့်
  • LEDs နှင့် current-limiting resistors သုံးပါ
  • Circuit ပြောင်းမတင်မီ power ဖြုတ်ပါ
  • လုံခြုံသော operation အတွက် USB power သုံးပါ
